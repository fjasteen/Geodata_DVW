# Geodata_DVW

To transform DVW geodata for the EWS

This project transforms the DVW provided geodata to data that is
suitable for the EWS. The data is set into a suitable CRS and is saved
in 4 folders:

1\. "./data/output/Districts/"

2\. "./data/output/Sector_extended/"

3\. "./data/output/Sector_core/"

4\. "./data/output/Sector_parcels/"
